myFlix

This web application will provide users with information on movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Request URL HTTP Method Response
Get a list of movies /movies GET Returns JSON list of movies
Get data about a movie /movies/:title GET Returns JSON data about a specific movie
Get data about a genre by name/title /genres/:genre GET Returns JSON data with movies of a specific genre
Get data about a director by name /directors/:directorName GET Returns JSON data about the director
Allow new users /users/:newUser POST Adds new user to JSON object
Allow users to update their user info /users/:username PUT Returns the updated data fields
Allow users to add a movie to their list of favorites /favorites/:movieName POST Returns text confirming a movie has been added successfully
Allow users to remove a movie from their list of favorites /favorites/:deleteMovie Delete Returns text confirming a movie has been removed successfully
Allow existing users to deregister /users/:deleteAccount DELETE Returns text confirming the user has been deregistered
